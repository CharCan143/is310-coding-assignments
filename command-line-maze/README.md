# **The Old Abandoned Arcade** by **Charles Cantrell**

## Instructions:
Navigate the files using the below commands to find the solution.txt file with your terminal.

- **IMPORTANT:** Windows powershell user must run the following command before starting the maze once you have unzipped the maze: .\hide-dotfiles.ps1
- *The file that coincides this command is in in the "the-abandon-arcade" directory.*

## *Goal:*
Explore and find an arcade game of legend, Polybius - Long lost and forgotten.

## Relevant Commands: MacOs
- **cd [folderName]** : Change into a directory.
- **cd ..** : Go back a directory.
- **ls** : List of directory contents.
- **ls -la** : List of contents including hidden files.
- **cat [fileName]** : displays contents of a file.

## Relevant Commands: Windows
- **cd [folderName]** : Change into a directory.
- **cd ..** : Go back a directory.
- **ls** or **Get-ChildItem** : List of directory contents.
- **ls -force** or **Get-ChildItem -force** : List of contents including hidden files.
- **cat [fileName]** : displays contents of a file.
